// các câu lệnh để tạo
- install nestjs: npm i -g @nestjs/cli
- create project: nest new `project`
- run: npm run start:dev    
mặc định sẽ chạy trên cổng 3000.

// tạo thư mục
- nest g module `users`     --> create folder user
- nest g controller `users` --no-spec   --> create controller file 
- nest g service `users` --no-spec      --> create service file
